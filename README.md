# FastAPI Notebook MVP

Простой веб-блокнот на FastAPI для локального использования.

## Возможности

- Создание страниц блокнота
- Переименование страниц
- Редактирование содержимого
- Удаление страниц
- Простейший HTML-интерфейс
- Хранение данных в JSON (без БД)
